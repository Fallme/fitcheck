# 💪 FitCheck - 智能锻炼打卡

一款现代化的个人健身打卡 Web 应用，苹果风格界面，专注于徒手和哑铃训练。

## 在线体验

直接打开 `index.html` 即可使用，无需安装任何依赖。

## 功能特色

### 📋 打卡清单式交互
- 打开即显示当天训练计划，一键开始锻炼
- 分组圆点逐组打卡，完成自动打勾 ✓
- 计时任务内嵌倒计时，支持暂停/跳过/重来
- 计时完成后自动进入休息倒计时，无缝衔接下一组

### 🎯 5种训练重点可选
| 重点 | 说明 |
|------|------|
| 手臂 | 二头肌、三头肌为主，穿插肩部和有氧 |
| 胸部 | 胸大肌为主，配合三头肌和肩部 |
| 腿部 | 股四头肌、臀部为主，搭配核心训练 |
| 肩部 | 三角肌为主，配合手臂和有氧 |
| 全身 | 推拉腿分化训练，均衡发展 |

### 📊 数据统计与图表
- 顶部实时显示完成度百分比、预计消耗卡路里
- 折线图展示每日完成率趋势（近7天/近30天）
- 饼状图展示锻炼部位分布
- 连续打卡天数、总消耗 kcal 等核心指标

### 🎉 成就激励
- 每日完成 100% 自动触发彩带动效
- 历史记录自动保存，支持导出 .md 数据文件

### 🍎 苹果风格设计
- iOS 风格毛玻璃效果和圆角卡片
- SF Pro 系统字体，流畅动画过渡
- 移动端优先，完美适配手机屏幕

### 💧 营养追踪
- 饮水目标追踪（杯数）
- 蛋白质摄入追踪（克数）
- 目标可自定义，进度条实时显示

### ⚙️ 3档强度调节
| 模式 | 特点 |
|------|------|
| 轻松 | 少量组数，适合新手入门 |
| 标准 | 适中训练量，日常锻炼 |
| 强化 | 增加组数和次数，进阶挑战 |

## 训练计划设计原则

基于主流健身训练科学：
- **复合动作优先**：深蹲、俯卧撑等多关节动作排在前面
- **合理休息安排**：训练日与休息日交替，同一肌群间隔48小时
- **渐进超负荷**：通过强度调节实现训练量递增
- **每日15-22分钟**：高效短时训练，适合忙碌的现代人

## 技术栈

- 纯 HTML + CSS + JavaScript，零依赖
- Chart.js 4.x 图表可视化
- SVG 动画小人动作图解
- LocalStorage 本地数据持久化
- 响应式设计，PWA-ready
